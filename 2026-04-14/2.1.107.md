# Claude Code 2.1.107 实用技巧 - 2026-04-14

## 概述

v2.1.107 是一个小型维护版本，于 2026 年 4 月 14 日发布，主要改善了长时间操作时的用户体验。

## 技巧 1：更早显示思考进度提示

**来源：** [Claude Code 官方更新日志](https://code.claude.com/docs/en/changelog) | [GitHub Releases](https://github.com/anthropics/claude-code/releases)

### 摘要

v2.1.107 改进了扩展思考指示器，在长时间操作期间更早地显示旋转进度提示，让用户能更清楚地了解 Claude 的思考进度。

### 关键要点

- **改进内容**：当 Claude 进行复杂推理时，进度提示会更早出现，而不是让用户长时间面对空白等待
- **适用场景**：在处理大型代码库分析、复杂架构决策或深度调试时，此改进特别有用
- **用户体验**：旋转进度动画让你知道 Claude 正在积极思考，而非卡住

### 实用建议

如果你使用 Claude Code 处理复杂任务时经常需要等待较长时间，升级到 v2.1.107+ 可以获得更好的等待体验。配合 `/effort` 命令可以调整推理深度：

```bash
# 设置高推理深度，适合复杂架构决策
/effort high

# 设置低推理深度，适合简单的代码修改
/effort low
```

## 技巧 2：善用等待时间查看详细日志

**来源：** [Claude Code 命令速查表](https://computingforgeeks.com/claude-code-cheat-sheet/)

### 摘要

在 Claude 进行长时间思考时，你可以利用 `Ctrl+O` 查看详细的运行日志（transcript），了解 Claude 正在执行的具体操作。

### 关键要点

- 按 `Ctrl+O` 可以切换到详细日志视图
- 在 v2.1.108 中还新增了 "verbose" 标识，更清楚地指示你正在查看详细日志
- 这在等待长时间操作时尤其有用，帮助你了解 Claude 的执行进度

```bash
# 在 Claude 运行过程中按下快捷键查看日志
Ctrl+O  # 切换详细日志视图
```
